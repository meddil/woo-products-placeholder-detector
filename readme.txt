This plugin detects WooCommerce products with placeholder images or texts.

== Description ==
Detects and flags products with:
- Placeholder featured images
- Placeholder or "Lorem ipsum" text in title/description
Useful for WooCommerce store admins to clean up their catalog.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
